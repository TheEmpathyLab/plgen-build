## Decisions made

- **Web form over curl** — target audience (writers, journalists) won't use command line
- **Claude Project over prompt-per-session** — persistent instructions, no setup friction
- **sessionStorage for API key** — persists within browser session, clears on close
- **shared.css as single source of truth** — all pages must use it, no new fonts or variables
- **`plgen-build` repo** — document the build process publicly as a demonstration of PLGen
